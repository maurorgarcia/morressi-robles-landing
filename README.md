# Morressi Robles - Bienes Raíces

Sitio web y buscador inteligente de propiedades premium de Morressi Robles.

## Características

- Buscador inteligente de propiedades.
- Filtrado por operación (Venta/Alquiler), Zona, Tipo de propiedad y Presupuesto.
- Ficha detallada de propiedades con galería e información del asesor.
- Integración con base de datos de contacto (Supabase).
- Enlaces integrados a Instagram y ubicación en Google Maps.

## Desarrollo Local

1. Instalar dependencias:
   ```bash
   npm install
   ```

2. Correr el servidor de desarrollo:
   ```bash
   npm run dev
   ```
